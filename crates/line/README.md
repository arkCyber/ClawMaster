# clawmaster-line

Line Messaging API integration for ClawMaster.

## Features
- ✅ Line Messaging API support
- ✅ Push messages
- ✅ Reply messages
- ✅ Rich menus
- ✅ Flex messages

## Configuration
```toml
[channels.line]
enabled = true
channel_access_token = "your_token"
channel_secret = "your_secret"
```
