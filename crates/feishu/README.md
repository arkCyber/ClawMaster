# clawmaster-feishu

Feishu/Lark (飞书) integration for ClawMaster.

## Features
- ✅ Feishu Open API support
- ✅ Message sending
- ✅ Interactive cards
- ✅ Bot events
- ✅ Access token management

## Configuration
```toml
[channels.feishu]
enabled = true
app_id = "cli_xxx"
app_secret = "your_secret"
verification_token = "your_token"
encrypt_key = "your_key"
```
