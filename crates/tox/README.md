# clawmaster-tox

Tox protocol integration for ClawMaster.

## Features
- ✅ Tox protocol support
- ✅ P2P messaging
- ✅ End-to-end encryption
- ✅ Decentralized

## Configuration
```toml
[channels.tox]
enabled = true
tox_id = "your_tox_id"
bootstrap_nodes = ["node1.tox.chat:33445", "node2.tox.chat:33445"]
save_file = "/path/to/save.tox"
```
