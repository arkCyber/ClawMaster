# clawmaster-matrix

Matrix protocol integration for ClawMaster.

## Features
- ✅ Matrix Client-Server API
- ✅ Room management
- ✅ Message sync
- ✅ E2E encryption (optional)
- ✅ Federation support

## Configuration
```toml
[channels.matrix]
enabled = true
homeserver_url = "https://matrix.org"
user_id = "@bot:matrix.org"
access_token = "your_token"
rooms = ["!room1:matrix.org"]
```
