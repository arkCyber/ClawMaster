## ⚠️ 发现的问题

### 1. "No LLMs Connected" 提示
**问题**: 截图显示 "No LLMs Connected"  
**原因**: 未配置 LLM 提供商  
**解决方案**: 
1. 点击 "Settings" 按钮
2. 导航到 "LLM Providers" (或直接访问 `/settings/llms`)
3. 添加至少一个 LLM 提供商的 API 密钥

**配置步骤**:
```
1. 访问 https://localhost:59233/settings/llms
2. 选择提供商（如 OpenAI）
3. 输入 API 密钥
4. 点击 "Save" 或 "Test Connection"
5. 返回聊天页面
```

---
