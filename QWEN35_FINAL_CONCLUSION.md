## 🔧 解决方案

### 要让工具真正执行，需要：

1. **修复 Podman 沙箱配置**
   ```bash
   # 检查 Podman 状态
   podman machine list
   
   # 确保 Podman 运行
   podman machine start
   
   # 测试 Podman
   podman run hello-world
   ```

2. **或者禁用沙箱**（不推荐）
   在 `~/.config/clawmaster/clawmaster.toml` 中：
   ```toml
   [tools.exec.sandbox]
   mode = "none"  # 禁用沙箱
   ```

3. **测试不需要沙箱的工具**
   - calc - 数学计算
   - time - 时间查询  
   - news_search - 新闻搜索
   - web_search - 网页搜索
