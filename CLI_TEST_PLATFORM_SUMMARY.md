## 🚀 使用方法

### 快速开始

```bash
# 进入测试平台目录
cd cli_test_platform

# 运行演示
./demo.sh

# 或直接运行特定测试
./interactive_test.sh      # 交互式测试
./auto_test.sh            # 自动化测试
./performance_test.sh     # 性能测试
./log_analyzer.sh         # 日志分析
```
