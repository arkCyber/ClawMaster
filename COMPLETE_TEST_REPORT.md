## 🔍 发现的问题

### 问题 1: 身份问答误触发工具调用

**严重程度**: 中等 ⚠️

**现象**:
- 用户问 "你是谁？"
- 模型调用 `news_search` 工具，参数为 `{"query": "arkSong"}`
- 没有直接回答身份

**根本原因**:
1. 系统提示词中强调了 "arkSong" 身份
2. 模型将 "arkSong" 理解为需要搜索的关键词
3. 缺少明确的 "身份问答不要调用工具" 指令

**影响**:
- 用户体验差
- 浪费 API 调用
- 响应不符合预期

**修复状态**: ✅ 已修复

**修复方案**:
在系统提示词中添加了明确的身份问答规则：
```
**IDENTITY QUESTIONS - DO NOT USE TOOLS**:
When user asks about YOUR identity (你是谁/who are you/what are you):
- DO NOT call any tools
- Respond DIRECTLY in the user's language
- Say: "我是 arkSong，一个有工具调用能力的助手"
- NEVER search for "arkSong" in news or web
```

**修复位置**: `crates/agents/src/prompt.rs:396-401`

**验证**: 需要重新测试 "你是谁？"
