#### 性能问题 ⚠️

1. **事件处理器串行执行**
   - 影响: 多个处理器时延迟累加
   - 建议: 并行执行独立处理器

2. **配置频繁写入**
   - 影响: I/O 开销
   - 建议: 批量提交

3. **依赖解析重复计算**
   - 影响: CPU 浪费
   - 建议: 添加缓存

4. **注册表线性搜索**
   - 影响: 大量插件时性能下降
   - 建议: 添加索引

## 💡 使用指南

### 快速开始

```bash
# 1. 创建新插件项目
clawmaster-dev init my-plugin --type plugin

# 2. 进入项目
cd my-plugin

# 3. 查看项目结构
tree .

# 4. 构建项目
clawmaster-dev build

# 5. 运行测试
clawmaster-dev test

# 6. 验证项目
clawmaster-dev validate

# 7. 生成文档
clawmaster-dev docs --open
```

### 创建 Skill

```bash
# 在现有项目中

clawmaster-dev init my-skill --type skill
```
