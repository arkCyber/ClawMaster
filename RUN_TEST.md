# 🚀 运行自动测试

## 快速开始

在终端运行：

```bash
cd /Users/arksong/ClawMaster
./simple_auto_test.sh
```

## 测试流程

1. **脚本启动** - 检查系统状态
2. **提示测试1** - 要求您在 WebUI 输入 "美国新闻"
3. **自动监控** - 脚本监控日志文件，检测新活动
4. **自动分析** - 分析日志中的工具调用情况
5. **显示结果** - 显示测试是否通过
6. **继续测试2** - 测试 calc 工具（对照组）
7. **生成报告** - 保存详细测试报告

## 如果测试失败

脚本会显示失败原因，例如：

```
  ❌ 测试失败: 美国新闻
  ❌ 未检测到工具调用 (tool_calls_count=0)
  ❌ 响应包含禁止的文本
```

## 查看详细报告

```bash
cat /tmp/test_report_*.txt
```

## 实时监控日志（可选）

在另一个终端窗口：

```bash
tail -f /tmp/clawmaster_final.log | grep -E "(tool_calls_count|news_search|tool_call)"
```
