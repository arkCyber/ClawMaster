## 🔍 日志分析要点

### 关键日志模式

#### 工具调用成功
```
INFO executing tool tool=<工具名> id=<ID> args={...}
```

#### 参数提取
```
INFO <工具名>: <参数信息>
```

#### 结果返回
```
INFO tool result: <结果>
```

#### 错误信息
```
ERROR <错误描述>
WARN <警告描述>
```

#### 性能指标
```
INFO streaming LLM response complete iteration=<N> input_tokens=<N> output_tokens=<N>
```

## 🎯 成功标准

### 每个测试的成功标准

1. **工具调用** ✅
   - 调用了预期的工具
   - 工具名称正确

2. **参数提取** ✅
   - 参数正确提取
   - 参数类型正确
   - 参数值正确

3. **结果返回** ✅
   - 返回了有效结果
   - 结果格式正确
   - 结果内容准确

4. **错误处理** ✅
   - 没有意外错误
   - 错误信息清晰（如有）

5. **性能指标** ✅
   - 迭代次数合理（< 10）
   - 响应时间可接受
   - Token 使用合理
