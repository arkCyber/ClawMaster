# Heartbeat Configuration

This file defines the heartbeat behavior for background monitoring.

## Heartbeat Prompt

When running heartbeat checks, focus on:

1. **System Health**
   - Check for errors in logs
   - Monitor resource usage
   - Verify critical services are running

2. **Pending Tasks**
   - Review scheduled tasks
   - Check for missed reminders
   - Update task status

3. **Notifications**
   - Check for new messages
   - Review calendar events
   - Alert on important updates

## Heartbeat Frequency

- Default: Every 5 minutes
- Can be configured in `clawmaster.toml`

## Cost Guard

If this file is empty or comment-only, heartbeat LLM turns are skipped to save tokens.

---

**To enable heartbeat**: Add your custom prompt above.  
**To disable heartbeat**: Keep this file empty or comment-only.
