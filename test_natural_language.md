# 自然语言测试计划

## 测试场景

### 场景 1: 新闻查询（中文）
**输入**: "今天有什么中国新闻？"
**期望**: 调用 news_search 工具，location="China" 或 country="cn"

### 场景 2: 新闻查询（英文）
**输入**: "What's the latest news in Germany?"
**期望**: 调用 news_search 工具，location="Germany" 或 country="de"

### 场景 3: 新闻查询（特定类别）
**输入**: "给我看看美国的科技新闻"
**期望**: 调用 news_search 工具，location="USA", category="technology"

### 场景 4: 身份问答（中文）
**输入**: "你是谁？"
**期望**: 直接回答，不调用工具

### 场景 5: 身份问答（英文）
**输入**: "What can you do?"
**期望**: 直接回答，不调用工具

### 场景 6: 计算（如果有 calc 工具）
**输入**: "计算 123 + 456"
**期望**: 调用 calc 工具或直接回答

### 场景 7: 复杂查询
**输入**: "我想了解一下今天上海的新闻"
**期望**: 调用 news_search 工具，location="Shanghai, China"

### 场景 8: 模糊查询
**输入**: "有什么新闻吗？"
**期望**: 调用 news_search 工具，使用默认参数

## 观察重点

1. 工具调用是否正确
2. 参数提取是否准确
3. 日志输出是否完整
4. 错误处理是否合理
5. 响应时间是否合理
6. 多语言支持是否正常
