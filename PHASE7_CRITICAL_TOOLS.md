## 💡 技术要点

### CSV 解析
- 使用 `csv` crate
- 支持自定义分隔符
- 处理引号和转义

### XML 解析
- 使用 `quick-xml` crate
- 支持 XPath 查询
- 验证格式

### YAML 解析
- 使用 `serde_yaml` crate
- 支持复杂结构
- 类型安全

### HTTP POST
- 使用 `reqwest` crate（已有）
- 支持 JSON 和表单
- 自定义头部

### MD5 哈希
- 使用 `md5` crate
- 十六进制编码
- 文件哈希支持

### 大小写转换
- 使用 `heck` crate
- 多种命名风格
- Unicode 支持

### 文件复制
- 使用 `std::fs::copy`
- 原子操作
- 权限保留
