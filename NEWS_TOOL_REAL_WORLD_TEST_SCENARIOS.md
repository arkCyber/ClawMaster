## 🎯 测试场景设计

### 场景 1: 中国二线城市新闻

**用户对话**:
```
用户: "成都最近有什么美食新闻吗？"
```

**预期行为**:
1. LLM 提取位置: "Chengdu, China"
2. `parse_location()` 返回: "cn"
3. 查询 8 个中国新闻源
4. 返回成都相关美食新闻

**测试点**:
- ✅ LLM 智能位置提取
- ✅ 中文城市名识别
- ✅ 关键词过滤（美食）
- ✅ 多源聚合
