### 1. 问题根源 ✅ 已找到

**根本原因**: Podman 沙箱配置不完整

## 🔧 推荐配置

### 生产环境配置

**文件**: `~/.config/clawmaster/clawmaster.toml`

```toml
# Ollama Provider 配置
[providers.ollama]
base_url = "http://localhost:11434/v1"
models = ["qwen3.5:9b"]
enabled = true

# 工具配置
[tools.exec.sandbox]
mode = "none"  # 禁用 Podman，使用 WASM
scope = "session"
workspace_mount = "ro"
home_persistence = "shared"
no_network = false
```

**关键点**:
- ✅ 简单配置，无需额外依赖
- ✅ WASM 工具自动启用
- ✅ 工具可以正常执行
