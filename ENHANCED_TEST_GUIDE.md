### 生成的文件

测试完成后会在 `enhanced_test_YYYYMMDD_HHMMSS/` 目录下生成：

1. **summary.txt** - 测试摘要
   ```
   PASS: calc - 基础加法 (2s)
   PASS: calc - 基础减法 (1s)
   FAIL: web_fetch - 获取 JSON API
   SKIP: sessions_history - 需要活动会话
   ```

2. **detailed.log** - 详细日志
   ```
   === Test #1: calc ===
   Description: 基础加法
   Command: echo '2 + 2' | cargo run --bin clawmaster -- tools exec calc
   Duration: 2s
   Exit Code: 0
   Output:
   4
   ```

3. **failures.txt** - 失败详情
   ```
   [失败测试的完整输出]
   ```

4. **TEST_REPORT.md** - Markdown 报告
   - 测试统计
   - 测试覆盖
   - 通过/失败/跳过详情
   - 结论
