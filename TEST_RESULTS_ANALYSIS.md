## 🔍 发现的问题

### 问题 1: 身份问答误触发工具调用

**现象**: 
- 输入 "你是谁？"
- 模型调用了 `news_search` 工具，参数为 `{"query": "arkSong"}`

**根本原因**:
- 系统提示词中强调了 "arkSong" 身份
- 模型错误地将 "arkSong" 理解为新闻搜索关键词
- 缺少明确的"不要为身份问答调用工具"的指令

**影响**: 
- 用户体验差
- 浪费 API 调用
- 响应不符合预期

## 🛠️ 需要修复的问题
