# ClawMaster CLI 测试平台

完善的命令行测试平台，无需启动 WebUI 即可进行全面的功能测试。

## 🎯 功能特性

### 1. 交互式测试
- 实时对话测试
- 流式输出支持
- 工具调用观察

### 2. 自动化测试
- 预定义测试场景
- 批量测试执行
- 结果自动分析

### 3. 性能测试
- 响应时间统计
- Token 使用分析
- 迭代次数监控

### 4. 日志分析
- 实时日志查看
- 关键信息提取
- 错误自动检测

## 📋 测试场景

### 新闻查询测试
- 中文新闻查询
- 英文新闻查询
- 特定类别查询
- 地理位置查询

### 身份问答测试
- 中文问答
- 英文问答
- 能力查询

### 计算功能测试
- 简单计算
- 复杂表达式
- 数学函数

### 多语言测试
- 中英混合
- 多语言切换
- 语言识别

## 🚀 快速开始

### 交互式测试
```bash
./cli_test_platform/interactive_test.sh
```

### 自动化测试
```bash
./cli_test_platform/auto_test.sh
```

### 性能测试
```bash
./cli_test_platform/performance_test.sh
```

## 📊 测试报告

测试完成后会生成：
- 测试结果摘要
- 性能统计数据
- 错误日志分析
- 改进建议
