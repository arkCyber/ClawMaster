## 🎯 WASM 工具特性

### 技术架构

| 组件 | 说明 |
|------|------|
| **WasmComponentEngine** | Wasmtime 引擎封装 |
| **WasmToolRunner** | AgentTool 适配器 |
| **WasmResourceLimiter** | 资源限制器 |
| **EpochTicker** | 超时控制 |
| **CachingWasmToolRunner** | 结果缓存 |

### 安全特性

- ✅ **燃料限制**: 防止无限循环
- ✅ **内存限制**: 防止内存耗尽
- ✅ **超时控制**: Epoch-based 超时
- ✅ **SSRF 防护**: HTTP 工具的 SSRF 检查
- ✅ **沙箱隔离**: 完全隔离执行环境

### 资源配置

```rust
// calc 工具配置
fuel: 100,000
memory: 2 MB
timeout: 2 秒

// web_fetch/web_search 工具配置
fuel: 1,000,000
memory: 16 MB
timeout: 30 秒 (可配置)
cache_ttl: 60 分钟 (可配置)
```
