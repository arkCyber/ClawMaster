## 🎨 UI 设计特性（DO-178C 标准）

### **1. 暖色调配色系统**
```rust
// 暗模式 - 温暖的深棕色系
DARK_BG:          #1c1814  // 深棕背景
DARK_SURFACE:     #2a2420  // 卡片背景
DARK_BORDER:      #3d3530  // 边框
TEXT_PRIMARY:     #f4e9dc  // 主要文本（暖白）
TEXT_SECONDARY:   #c9b89a  // 次要文本
ACCENT:           #d4946f  // 强调色（暖橙）
SUCCESS:          #8fbc8f  // 成功（暖绿）
WARNING:          #daa520  // 警告（金色）
ERROR:            #cd5c5c  // 错误（暖红）
```

### **2. DO-178C Level A 合规性**

#### **安全性要求**
- ✅ **错误状态清晰可见** - 使用颜色编码和图标
- ✅ **关键操作确认** - 危险按钮使用警告色
- ✅ **状态转换可追溯** - 所有变更有日志
- ✅ **一致性** - 所有页面使用统一的布局和控件

#### **可用性要求**
- ✅ **明确的标签** - 所有控件都有描述性文本
- ✅ **视觉层次** - 使用字体大小和颜色区分重要性
- ✅ **响应式反馈** - 按钮悬停、点击状态
- ✅ **可访问性** - 高对比度、清晰的视觉提示

#### **健壮性要求**
- ✅ **优雅降级** - 后端断开时使用模拟数据
- ✅ **错误恢复** - 所有错误都有恢复路径
- ✅ **资源管理** - 滚动列表、懒加载支持
